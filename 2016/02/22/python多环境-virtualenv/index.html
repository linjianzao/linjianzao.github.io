Python多环境:virtualenv

安装:

pip install virtualenv

创建:

mkdir djan_env
cd djan_env
source ./bin/activate

退出:

deactivate

注意:

如果你是在非sudo环境下创建virtualenv，
那sudo的时候用的还是真是环境
